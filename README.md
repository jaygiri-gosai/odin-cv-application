# CV Application

A CV Builder application developed using React as part of The Odin Project curriculum.

## Overview

This application allows users to create and manage a CV by entering personal, educational, and professional information. After submission, each section can be edited independently without affecting the rest of the CV.

## Features

### General Information

- Name
- Email Address
- Phone Number

### Educational Experience

- School Name
- Title of Study
- Date of Study

### Practical Experience

- Company Name
- Position Title
- Responsibilities
- Employment Start Date
- Employment End Date

### Additional Functionality

- Submit and display CV information
- Edit individual sections after submission
- Form validation using HTML attributes
- Responsive and user-friendly layout

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd cv-application
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
